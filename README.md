# 🛍️ ShopMod

**ShopMod** — это полнофункциональная платформа на Django для управления товарами и контентом с продвинутой системой прав доступа.

## 🚀 Возможности

### 📦 Управление товарами
- **Полный CRUD** для товаров с изображениями и описаниями
- **Система владения** - каждый товар привязан к владельцу
- **Модерация контента** - публикация/снятие с публикации
- **Валидация данных** - цена, запрещенные слова, форматы изображений

### 👥 Система пользователей  
- **Кастомная модель User** с email-аутентификацией
- **Регистрация и авторизация** с приветственными письмами
- **Управление профилем** с аватарами и контактной информацией
- **Группы и права доступа** - гибкая система разрешений

### 📝 Блог-платформа
- **Публикация статей** с rich-контентом и изображениями
- **Система статусов** - черновики и опубликованные статьи
- **Контент-менеджеры** - отдельная группа для управления блогом
- **Комментирование** и взаимодействие с контентом

### 🛡️ Безопасность и права доступа
- **Группа "Модератор продуктов"** - управление публикацией и удаление товаров
- **Группа "Контент-менеджер"** - полный контроль над блогом
- **Владельческий доступ** - редактирование только своих товаров/статей
- **Защита от спама** - валидация запрещенных слов

## 👥 Группы пользователей и права

### 🔧 Модератор продуктов
- **Права:** `can_unpublish_product`, `delete_product`
- **Возможности:**
  - Снятие товаров с публикации
  - Публикация скрытых товаров  
  - Удаление любых товаров
  - Просмотр всех товаров (включая скрытые)

### ✍️ Контент-менеджер
- **Права:** `can_publish_blog`, `can_manage_blog` + CRUD
- **Возможности:**
  - Создание и редактирование статей
  - Публикация и снятие статей
  - Удаление любых статей
  - Полный контроль над блогом

### 👤 Обычный пользователь
- **Возможности:**
  - Создание товаров (становится владельцем)
  - Редактирование только своих товаров
  - Удаление только своих товаров
  - Просмотр опубликованных товаров и статей

### 🎭 Анонимный пользователь
- **Возможности:**
  - Просмотр опубликованных товаров
  - Просмотр опубликованных статей блога
  - Регистрация и вход в систему

## ⚙️ Установка и запуск

### 1. Клонирование и настройка
```bash
git clone <repository-url>
cd shopmod
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```
### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```
### 3. Скопируйте `.env.example` в `.env`:
```bash
cp .env.example .env
```
### 4. База данных и миграции
```bash
python manage.py makemigrations
python manage.py migrate
```
### 5. Настройка групп и прав
```bash
python manage.py setup_moderator_group
python manage.py setup_content_manager_group
```
### 6. Создание суперпользователя
```bash
python manage.py createsuperuser
```
### 7. Запуск сервера
```bash
python manage.py runserver
```

## 🔧 Технологический стек
 - Backend: Python 3.12, Django 5.2
 - Database: PostgreSQL
 - Frontend: Bootstrap 5, HTML5, CSS3
 - Authentication: Django Auth с кастомной моделью
 - Media: Pillow для работы с изображениями
 - Code Quality: Black, isort
 - Environment: python-dotenv
