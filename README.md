# 🛍️ ShopMod
![Django](https://img.shields.io/badge/Django-5.2-green)
![Python](https://img.shields.io/badge/Python-3.12-blue)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5-purple)
**ShopMod** — это учебный проект на Django, реализующий систему управления товарами с полным CRUD функционалом и встроенной контент-модерацией.

## 🚀 Возможности

### 📦 Управление товарами
- **Создание** товаров с валидацией данных
- **Просмотр** списка всех товаров  
- **Редактирование** существующих товаров
- **Удаление** товаров с подтверждением

### 🛡️ Валидация и безопасность
- **Защита от спама** — проверка на запрещенные слова в названиях и описаниях
- **Валидация цены** — цена не может быть отрицательной
- **Проверка изображений** — поддержка только JPEG/PNG форматов, макс. размер 5 МБ

### 🎨 Пользовательский интерфейс
- **Адаптивный дизайн** на Bootstrap 5
- **Профессиональные формы** с кастомной стилизацией
- **Оптимальное отображение** изображений (формат 4:3)
- **Интуитивная навигация** между разделами

## 🛠️ Технологический стек

- **Backend**: Python 3.12, Django 5.2
- **Database**: PostgreSQL
- **Frontend**: Bootstrap 5, HTML5, CSS3
- **Дополнительно**: django-forms, pillow, python-dotenv

## ⚙️ Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/your-username/shopmod.git
cd shopmod
```
### 2. Создание виртуального окружения
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```
### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```
### 4. Настройка окружения, скопируйте `.env.example` в `.env`:
```bash
cp .env.example .env
```

### 5. Настройка базы данных
```bash
python manage.py makemigrations
python manage.py migrate
```
### 6. Запуск сервера
```bash
python manage.py runserver
```
Приложение будет доступно по адресу: http://127.0.0.1:8000/products/

## 🔧 Кастомная валидация
### Запрещенные слова
Система блокирует использование следующих слов в названиях и описаниях:

 - казино, криптовалюта, крипта, биржа
 - дешево, бесплатно, обман, полиция, радар

### Валидация изображений
 - Форматы: JPEG, PNG 
 - Максимальный размер: 5 МБ 
 - Проверка MIME-type для безопасности
