# ShopMod
Учебный проект на Django, реализующий классическую систему CRUD (Create, Read, Update, Delete) для управления товарами с встроенной системой валидации контента. Проект автоматически фильтрует и блокирует попытки добавления товаров с запрещенными словами в названиях и описаниях.

Ключевые особенности:

Полный цикл CRUD: Удобное добавление, просмотр, редактирование и удаление товаров.

Защита от спама: Встроенная валидация форм, проверяющая отсутствие списка запрещенных слов (например, "казино", "криптовалюта", "бесплатно").

Независимость от регистра: Фильтр срабатывает на слова в любом регистре (например, "КаЗиНо" тоже будет заблокировано).

Чистая архитектура: Использование Django Forms и ModelForms для строгого разделения логики, данных и представления.

Технологический стек:

Backend: Python, Django

База данных: SQLite (по умолчанию, для разработки)

Инструменты: Django ORM, Django Forms, Class-Based Views

Что реализовано:

Модель Product с полями "Название" и "Описание".

Кастомная форма ProductForm с валидацией на основе списка запрещенных слов.

Views на основе классов (ListView, CreateView, UpdateView, DeleteView).

Шаблоны для отображения списка товаров, форм создания/редактирования и подтверждения удаления.

Для кого этот проект:
Проект служит наглядным примером для тех, кто изучает Django и хочет понять работу с формами, валидацией данных и реализацией базовых операций с данными в веб-приложениях.
