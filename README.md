# 🛍️ ShopMod

**ShopMod** — это полнофункциональная платформа на Django для управления товарами, категориями и контентом с продвинутой системой прав доступа и многоуровневым кешированием.

## 🚀 Возможности

### 📦 Управление товарами
- **Полный CRUD** для товаров с изображениями и описаниями
- **Система владения** - каждый товар привязан к владельцу
- **Модерация контента** - публикация/снятие с публикации
- **Валидация данных** - цена, запрещенные слова, форматы изображений

### 🗂️ Система категорий
- **Иерархическая структура** категорий товаров
- **Slug-URL** для SEO-оптимизации
- **Автоматический подсчет** товаров в категориях
- **Визуальная навигация** по категориям

### 👥 Система пользователей  
- **Кастомная модель User** с email-аутентификацией
- **Регистрация и авторизация** с приветственными письмами
- **Управление профилем** с аватарами и контактной информацией
- **Группы и права доступа** - гибкая система разрешений

### 📝 Блог-платформа
- **Публикация статей** с rich-контентом и изображениями
- **Система статусов** - черновики и опубликованные статьи
- **Контент-менеджеры** - отдельная группа для управления блогом

### ⚡ Производительность и кеширование
- **Низкоуровневое кеширование** списков товаров и категорий
- **Кеширование страниц** детальных просмотров
- **Сервисный слой** с централизованной бизнес-логикой
- **Redis-кеширование** с инвалидацией при изменениях

### 🛡️ Безопасность и права доступа
- **Группа "Модератор продуктов"** - управление публикацией и удаление товаров
- **Группа "Контент-менеджер"** - полный контроль над блогом
- **Владельческий доступ** - редактирование только своих товаров/статей
- **Защита от спама** - валидация запрещенных слов

## 👥 Группы пользователей и права

### 🔧 Модератор продуктов
- **Права:** `can_unpublish_product`, `delete_product`
- **Возможности:**
  - Снятие товаров с публикации
  - Публикация скрытых товаров  
  - Удаление любых товаров
  - Просмотр всех товаров (включая скрытые)

### ✍️ Контент-менеджер
- **Права:** `can_publish_blog`, `can_manage_blog` + CRUD
- **Возможности:**
  - Создание и редактирование статей
  - Публикация и снятие статей
  - Удаление любых статей
  - Полный контроль над блогом

### 👤 Обычный пользователь
- **Возможности:**
  - Создание товаров (становится владельцем)
  - Редактирование только своих товаров
  - Удаление только своих товаров
  - Просмотр опубликованных товаров и статей

### 🎭 Анонимный пользователь
- **Возможности:**
  - Просмотр опубликованных товаров
  - Просмотр опубликованных статей блога
  - Регистрация и вход в систему

## ⚡ Система кеширования

### Низкоуровневое кеширование
```python
# Сервисные функции с кешированием
get_cached_product_list(user)           # Список товаров
get_products_by_category(slug, user)    # Товары по категории
get_products_count()                    # Количество товаров
```
Кеширование страниц
```python
# Декораторы кеширования представлений
@cache_page(60 * 15)    # 15 минут для детальных страниц
@cache_page(60 * 30)    # 30 минут для списков
```
Ключи Redis
```text
shopmod:1:product_list_anon        # Список для анонимных
shopmod:1:product_list_6           # Список для пользователя ID=6
shopmod:1:products_category_*      # Товары по категориям
shopmod:1:all_categories           # Все категории
shopmod:1:blog_posts               # Статьи блога
```

## 🎯 API Endpoints
Товары
```text
GET  /products/                     # Список товаров
GET  /products/<id>/               # Детали товара
GET  /products/categories/         # Список категорий
GET  /products/category/<slug>/    # Товары категории
POST /products/create/             # Создание товара
POST /products/<id>/update/        # Обновление товара
POST /products/<id>/delete/        # Удаление товара
POST /products/<id>/publish/       # Публикация товара
POST /products/<id>/unpublish/     # Снятие с публикации
```
Блог
```text
GET  /blog/                        # Список статей
GET  /blog/post/<id>/              # Детали статьи
POST /blog/post/create/            # Создание статьи
POST /blog/post/<id>/update/       # Обновление статьи
POST /blog/post/<id>/delete/       # Удаление статьи
POST /blog/post/<id>/publish/      # Публикация статьи
POST /blog/post/<id>/unpublish/    # Снятие с публикации
```
Пользователи
```text
GET  /users/profile/               # Профиль пользователя
POST /users/profile/edit/          # Редактирование профиля
POST /users/register/              # Регистрация
POST /users/login/                 # Авторизация
POST /users/logout/                # Выход
```
## ⚙️ Установка и запуск
1. Клонирование и настройка
```bash
git clone <git@github.com:Anton742-Tu/ShopMod.git>
cd shopmod
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```
2. Установка зависимостей

```bash
pip install -r requirements.txt
```
3. Настройка окружения
```bash
cp .env.example .env
```
# Отредактируйте .env файл с вашими настройками
4. База данных и миграции
```bash
python manage.py makemigrations
python manage.py migrate
```
5. Настройка групп и прав
```bash
python manage.py setup_moderator_group
python manage.py setup_content_manager_group
```
6. Создание суперпользователя
```bash
python manage.py createsuperuser
```
7. Запуск сервера
```bash
python manage.py runserver
```

## 📊 Производительность
Оптимизации
Кеширование запросов - снижение нагрузки на БД на 80%

Select related - избегание N+1 запросов

Пагинация - ограничение выдачи до 12 товаров на странице

Инвалидация кеша - актуальность данных при изменениях

Метрики
Время ответа: ~50мс (с кешем) vs ~200мс (без кеша)

Нагрузка на БД: 1 запрос (с кешем) vs 5+ запросов (без кеша)

Память Redis: ~1-5MB в зависимости от количества товаров
